# IP Discovery Demo

Research IP discovery pipeline for a large agricultural research network.
Fetches real papers from the CGIAR CGSpace repository, filters by semantic similarity (sentence-transformers + FAISS), scores commercial potential, writes structured invention disclosures, and produces a self-contained HTML dashboard.

```
ip-discovery-demo/
│
├── run.py               ← the one script you run
├── dashboard.html       ← generated by run.py
├── database.db          ← SQLite, all results stored here
├── requirements.txt     ← libraries to install
└── README.md
```

## Run in 3 steps

**1. Install dependencies**
```bash
pip install -r requirements.txt
```

**2. Run the pipeline**
```bash
python run.py
```

**3. Open the dashboard**
```bash
open dashboard.html        # macOS
start dashboard.html       # Windows
xdg-open dashboard.html    # Linux
```

## What it does

| Step | What happens |
|---|---|
| **1 — Fetch** | Pulls 50 live papers from the CGSpace public REST API — titles, abstracts, authors, centre names |
| **2 — Filter** | Encodes abstracts with `all-MiniLM-L6-v2`; scores each against 5 innovation seed phrases via a FAISS flat inner-product index; keeps papers scoring ≥ 0.35 |
| **3 — Score** | Writes into the database: a commercial potential score (1–10), one-sentence reasoning, core invention description, three target organisation types |
| **4 — Disclose** | Top 5 papers by commercial score get a full structured invention disclosure: problem, technical approach, novelty, applications, licensing model |
| **5 — Dashboard** | Generates `dashboard.html` with all data embedded — searchable, sortable, expandable disclosure panels, no server required |

## Dashboard views

**View 1 — Pipeline overview:** Three numbers at the top — papers fetched, papers that passed the filter, disclosures drafted.

**View 2 — Candidates table:** Every paper that passed the filter with title, centre name, score out of 10, and invention summary. Sortable by score. Searchable by keyword.

**View 3 — Disclosure panel:** Click any row to expand a full invention disclosure with all six sections. Top-5 papers show a purple `DISCLOSURE` badge.

## Database schema

```sql
papers      (id, uuid, title, abstract, authors, collection, semantic_score)
scores      (id, paper_id, commercial_score, reasoning, invention_description, org_types)
disclosures (id, paper_id, invention_title, problem_solved, technical_approach,
             novelty, commercial_applications, licensing_model)
```

## Dependencies

```
sentence-transformers   all-MiniLM-L6-v2 model
faiss-cpu               vector similarity search
requests                HTTP client for CGSpace API
numpy                   array operations
```

No AI API keys required. All analysis is generated locally.
